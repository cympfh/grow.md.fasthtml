# grow.md.fasthtml

## What?

A markdown viwer.

## Why?

Re-implementation [grow.md](https://github.com/cympfh/grow.md.old) (which is broken) with FastHTML.

grow.md がぶっ壊れたので FastHTML を使って書き直してみた試作品

## Usage

```
$ chmod +x ./bin/grow
$ cd ~/Documents/
$ grow --port 8080
$ open http://localhost:8080
```
